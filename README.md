# Almeida & Associados Advocacia

Site institucional multipágina premium desenvolvido para o escritório de advocacia **Almeida & Associados**, especializado em Direito Societário e Tributário.

## 🛠️ Stack Tecnológica

- **Framework:** Astro
- **Estilização:** Tailwind CSS (Mobile-First)
- **Integração:** Resend (Notificações por e-mail)
- **Hospedagem:** Vercel (CI/CD via push no GitHub)

## 📐 Estrutura de Rotas

- `index.astro` — Home (Proposta de valor + Sócios + Áreas + Blog preview)
- `sobre.astro` — Nossa História / Perfil do Escritório
- `atuacao.astro` — Áreas de Especialidade
- `blog/index.astro` — Artigos informativos jurídicos
- `contato.astro` — Agendamento de reuniões e Formulário de contato

## 🚀 Como executar localmente

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
3. Acesse em seu navegador: `http://localhost:4321`
